# GreenCode

GreenCode is an API core system designed to support esm_greencode .
It consists of a **Spring Boot backend API** 

---

## 🚀 Features

### Backend (Spring Boot)
- RESTful API
- JWT/OAuth authentication
- PostgreSQL database support
- Centralised configuration (`config/`, `.env`)
- Dockerized for easy deployment
- Swagger/OpenAPI documentation


## 📁 Project Structure

The repository currently contains only the backend system.

- Backend: Core application logic and APIs
- Frontend: Not included in this repository (may be developed separately in the future)
